# Aura News - AI 驅動的新聞平台

## 產品介紹

Aura News 是一個革命性的全端新聞平台，結合了人工智慧技術與現代化網頁開發，為用戶提供智能化、個性化的新聞閱讀體驗。本平台採用前後端分離架構，具備 AI 自動新聞抓取與內容生成、響應式設計、SEO 優化等先進功能，是現代新聞媒體的理想解決方案。

### 核心價值主張

- **AI 驅動內容生成**：運用 Gemini AI 技術自動抓取、整理和生成高品質新聞內容
- **全端現代化架構**：Laravel + Vue 3 技術棧，確保系統穩定性和可擴展性
- **響應式用戶體驗**：完美適配桌面端和移動端，提供一致的閱讀體驗
- **SEO 自動化優化**：內建 SEO 工具，自動生成 sitemap 和 meta 標籤
- **自動化部署流程**：CI/CD 整合，支援一鍵部署和版本管理

## 後端功能詳述 (aura-news-backend)

### 🤖 AI 智能新聞處理系統

#### 自動新聞抓取
- **多源整合**：整合 NewsAPI.org 等多個新聞來源
- **智能分類**：自動將新聞歸類到對應分類（科技、政治、娛樂等）
- **內容清洗**：自動過濾無效內容、重複文章和垃圾信息
- **圖片驗證**：智能檢測和驗證新聞配圖的有效性

#### AI 內容生成與優化
- **Gemini AI 整合**：使用 Google Gemini 2.5 Flash Lite 模型
- **智能摘要生成**：自動生成 150 字精簡摘要
- **內容重寫**：將原始新聞重寫為 500 字完整文章
- **Markdown 格式**：支援富文本格式，提升閱讀體驗
- **多語言支援**：專為繁體中文內容優化

#### 背景任務處理
- **異步處理**：使用 Laravel Jobs 處理耗時的 AI 任務
- **錯誤重試機制**：自動重試失敗的處理任務
- **日誌記錄**：完整的處理過程追蹤和錯誤記錄

### 🔐 用戶認證與權限管理

#### 多層級權限系統
- **JWT/Token 認證**：安全的 API 認證機制
- **Laravel Sanctum**：現代化的 API 認證框架
- **角色權限控制**：支援管理員和一般用戶權限分離
- **會話管理**：安全的用戶會話處理

#### 安全防護
- **CORS 配置**：跨域請求安全控制
- **API 金鑰驗證**：外部 API 調用安全驗證
- **輸入驗證**：完整的數據驗證和清理機制

### 📊 內容管理系統

#### 文章管理
- **CRUD 操作**：完整的文章增刪改查功能
- **狀態管理**：草稿、發布、下架等狀態控制
- **批量操作**：支援批量文章管理
- **版本控制**：文章修改歷史追蹤

#### 分類管理
- **動態分類**：支援自定義新聞分類
- **分類關聯**：文章與分類的多對一關聯
- **SEO 優化**：分類頁面 SEO 友好設計

#### 媒體管理
- **圖片上傳**：支援多種圖片格式上傳
- **圖片處理**：自動圖片優化和縮放
- **CDN 整合**：支援 CDN 圖片加速

### 🛠 開發與部署工具

#### 開發環境
- **Laravel 12.x**：最新版本的 PHP 框架
- **PHP 8.x**：現代化 PHP 語言特性
- **MySQL/MariaDB**：可靠的關係型數據庫
- **Composer**：PHP 依賴管理

#### 測試與品質保證
- **PHPUnit 整合**：完整的單元測試和功能測試
- **自動化測試**：CI/CD 流程中的自動測試
- **代碼品質**：ESLint 和 Prettier 代碼格式化

#### 部署與維護
- **環境配置**：靈活的環境變數配置
- **數據庫遷移**：版本化的數據庫結構管理
- **種子數據**：預設數據自動填充
- **日誌管理**：完整的系統日誌記錄
